“i log my” is a web app for logging things. It currently has two modules; weight and calories.

Navigation

“i log my” at the top left takes you to the main page from where you can enter any of the modules.

“Weight” or “Calories” on the main page takes you to the relevant module page, and inserts the module name after “i log my”. From here you can choose any of the module related pages.

The Module name (top left, after “i log my”) always takes you to that module main page.
